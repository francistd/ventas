
1- archivos: contiene todas las imagenes de los productos agregados.
2- bd: carpeta que contiene los snipets y querys de la base de datos lista para ejecutar.
3- clases: contiene todas las clases de cada modulo, con sus respectivos metodos de crud
4- css: contiene los scripts css que serviran para dar estilo personalizado en caso de ser util.
5- img: contiene todas las imagenes que vamos a utilizar para agregar a las vistas.
6- js: contiene todos los scripts de js para cada modulo para ejecutar las funciones de crud.
7- librerias: contiene todas las librerias para la creacion de pdf,jquery,alertify,boostrap y select2.
8- procesos: contiene archivos intermedios entre las clases y las vistas, estos mandaran informacion entre una y otra cosa.
9- vistas: contiene todas las vistas de cada uno de los modulos.
